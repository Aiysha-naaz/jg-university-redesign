# JG University - Modern Landing Page Redesign

A high-performance, visually immersive university landing page built as part of the Coding Reducate.ai
Full Stack Development Internship Assignment. This project focuses on modern Full stack ,UI/UX principles, responsive design, and smooth user interactions.

## 🔗 Live Demo
[View Live Site](https://jg-university-redesign.vercel.app/)


## ✨ Features

### 1. Bento Grid Architecture
- Used a high-density "Bento" style layout for the Programs section to create a sophisticated visual hierarchy.
- Implemented `md:col-span` and `md:row-span` logic to give primary schools more visual weight.

### 2. Interactive UI/UX
- Orchestrated Animations: Powered by `framer-motion` for staggered entrance and hover effects.
- Micro-interactions: Hover states on program cards reveal specific course lists, reducing cognitive load.
- Apply Modal: A state-managed application form using React `useState` and Framer Motion's `AnimatePresence`.

### 3. Fully Responsive Design
- Mobile-first approach using Tailwind CSS.
- Interactive mobile navigation drawer for seamless browsing on smaller screens.

## 🛠️ Tech Stack

- Framework: [Next.js 14 (App Router)](https://nextjs.org/)
- Styling: [Tailwind CSS](https://tailwindcss.com/)
- Animations: [Framer Motion](https://www.framer.com/motion/)
- Icons: [Lucide React](https://lucide.dev/)
- Deployment: [Vercel](https://vercel.com/)

## 🚀 Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone [https://github.com/Aiysha-naaz/jg-university-redesign.git](https://github.com/Aiysha-naaz/jg-university-redesign.git)
   ```

2. Install dependencies:
```bash
   npm install
```

3. Run the development server:
```bash
   npm run dev
```


## 📸 Design Decisions
- Typography: Used a combination of bold headings for authority and slate-colored body text for readability.
- Color Palette: Utilized Indigo-600 as the primary brand color to evoke trust, paired with a clean white/slate background.
- Accessibility: Ensured high contrast ratios and semantic HTML tags for screen readers.
